# Idsec SDK

![Idsec SDK Golang](./media/sdk.png){: style="height:80%;width:70%"}

The official Golang SDK and CLI for CyberArk's Identity Security Platform Services.

## Motivation
Idsec SDK, an API first library, is used to automate different sets of operations in order to ease day to day development. Idsec provides a CLI for general purpose automation as well as a Golang SDK for code integration.
